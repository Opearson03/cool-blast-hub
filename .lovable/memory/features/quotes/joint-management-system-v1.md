# Memory: features/quotes/joint-management-system-v1
Updated: now

Expansion and control joints are managed via a multi-row interface where each configuration (depth, capping, dowels, caulking) supports manual or polyline plan measurement. Clicking 'Mark on Plans' triggers a takeoff workflow using a 4-part identifier ({parentScopeId}:{jointType}:joint:{jointId}) to ensure measured data is saved back to the correct parent scope (e.g., a specific driveway). Measurement uses the discrete point-pair mode. Expansion joints are 3m pieces (sticks) priced per piece — quantity is calculated as Total Length / 3, rounded up. Default price resolves via price key `EXJ{depth}30` (e.g., EXJ10030 = $95/piece for 100mm joints). The takeoff toolbar for joints displays context-specific instructions ("Tap to mark [jointLabel] lines") and a dedicated "Done" button that returns the user to the estimator module once segments are marked. Tool auto-activation occurs even if calibration is pending, triggering the calibration prompt automatically. Foam section is separate — foam is sold in 25m rolls, unchanged.
