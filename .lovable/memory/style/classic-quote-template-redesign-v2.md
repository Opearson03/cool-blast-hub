# Memory: style/classic-quote-template-redesign-v2
Updated: Today

The Classic quote template has been redesigned with a cleaner, more professional aesthetic. Key changes include:

## Page 1 Changes
- **Header**: Replaced heavy dark secondary-color background with a thin 4px primary-color accent bar + clean white header area with subtle bottom border
- **Info Boxes**: Removed boxed sections with dark headers; now uses primary-color left-border accents with more breathing room
- **Line Items Table**: Lighter gray-100 header instead of dark secondary color; rounded corners; integrated subtotal/GST rows in gray-50 background before the bold primary-color total row
- **Authorisation Block**: Larger signature lines (h-12), primary-color left accent border, bg-gray-50 background, increased text sizes for prominence

## Page 2 Changes
- **Header**: Matches Page 1 style exactly (thin accent bar + white background), removed negative margins that caused PDF rendering issues
- **Content Sections**: Replaced boxed sections with accent-border style matching Page 1; increased text from xs (10px) to sm (14px)
- **Authorisation Removed**: Duplicate signature block removed; simple footer directs user to sign on Page 1

## Design Philosophy
- Strategic use of primary color as accents (borders, total row) rather than solid blocks
- Secondary color is no longer used prominently, keeping the template lighter
- Consistent visual hierarchy with proper spacing between elements
- Both pages maintain independent A4 layout with 277mm min-height for Page 1
